Quick Start

1

Install

Install Adaptive with a single command:

curl -fsSL https://raw.githubusercontent.com/madhurprash/adaptive/main/scripts/install.sh | bash
2

Configure

Set up your environment with AWS credentials and observability platform API keys:

# AWS Configuration
AWS_REGION=us-east-1

# LangSmith or Langfuse
LANGSMITH_API_KEY=your_api_key_here
LANGFUSE_PUBLIC_KEY=your_public_key
3

Run

Start analyzing and optimizing your agents:

# Interactive mode
adaptive run

# With debug logging
adaptive run --debug

# Background daemon
adaptive daemon

Multi-Agent Architecture

The system uses specialized agents working together:

Insights Agent

Analyzes observability traces from LangSmith or Langfuse and generates insights about agent performance, error patterns, and optimization opportunities.

Evolution Agent

Transforms insights into concrete optimizations by proposing specific changes to system prompts with detailed diffs and approval workflows.

Routing Agent

Orchestrates workflow between agents, determining when evolution is needed based on user intent and the nature of generated insights.
